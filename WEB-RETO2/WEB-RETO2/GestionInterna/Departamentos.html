Department Management

In this section, you can manage departments.

Create Department - Allows you to create a new department in the system. You must fill in the corresponding fields.
Delete Department - Allows you to delete a department from the system. You must select the corresponding department.
Modify Department - Allows you to select and modify an existing department in the system.
Consult Department - Allows you to view detailed information about an existing department.
